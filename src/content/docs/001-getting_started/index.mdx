Ira Saas is a Next.js SaaS Starter that helps you build your own SaaS in minutes. It comes with a fully integrated Stripe billing system, a landing page, and a dashboard.

In this section, we learn how to install and run the SaaS kit on your local
machine.

Buckle up and let's get started!
